## 简介 (⁎˃ᆺ˂)

一个云中转盘(yunzhongzhuan.com, yzzpan.com)的第三方API，同时实现了云中转盘的WebDav只读挂载(?)

~~为什么是问号啊喂~~

## 用法 ʕ•͡-•ʔ

请您创建一个config.json，内容如下
``` config.json
{
  "USERNAME": "MY_USERNAME",
  "PASSWORD": "MY_PASSWORD"
}
```
然后安装requirements.txt里的需求，启动webdav.py 就可以了。

> 注意: 没有外链功能可能导致文件下载失败(这是一个已知Bug)
